Configuración compartida de la organización [AetherCode](https://github.com/aethercode).

Este repositorio centraliza recursos que aplican a todos los repos de la organización:

- [`profile/`](./profile) — Perfil público de la organización ([github.com/aethercode](https://github.com/aethercode)).
- `ISSUE_TEMPLATE/` — Plantillas de issues por defecto _(próximamente)_.
- `PULL_REQUEST_TEMPLATE.md` — Plantilla de pull requests _(próximamente)_.
- `workflows/` — GitHub Actions reutilizables _(próximamente)_.
- `CODEOWNERS` — Propietarios por defecto _(próximamente)_.
